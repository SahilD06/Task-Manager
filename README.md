# Task Mastery

A simple, beautifully minimal Task Manager application built as a full-stack project.

## Features
- **Frontend**: Extremely minimal and fast UI using React.
- **Backend API**: Standard REST endpoints using Next.js App Router API Routes (`GET`, `POST`, `PATCH`, `DELETE`).
- **Persistence**: File-based storage (`data/tasks.json`) ensures data persists across dev server restarts.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the development server**:
   ```bash
   npm run dev
   ```
3. **Open application**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Tech Stack
- React / Next.js (App Router)
- Vanilla Global CSS
- Node.js `fs/promises` for DB storage
