# Task Manager

Full-stack task management app.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Database Setup**:
   Create a `.env.local` file with your connection string:
   ```env
   MONGODB_URI=your_mongodb_uri
   ```

3. **Run**:
   ```bash
   npm run dev
   ```

## Stack
- Next.js
- MongoDB Atlas
- Vanilla CSS
